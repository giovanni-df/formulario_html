Formulário HTML Atividade

Formulário HTML Sala

a) Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

Serve para especificar uma dica curta que descreve o valor esperado de um campo de entrada.

b) Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

Quando presente, ele especifica que o elemento deve ser pré-selecionado (marcado) quando a página é carregada.

c) Na tag <input> para que serve o atributo "value"?

É usado para definir o valor inicial do campo de entrada. Isso significa que quando a página carrega, o campo de entrada terá o valor especificado pelo atributo "value".

d) Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

O atributo name da tag <input>.

e) Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

A tag <label> é usada para criar um rótulo ou etiqueta associada a um elemento do formulário.

O atributo "for" em um elemento <label> é usado para vincular o rótulo ao elemento de formulário associado.

f) Na tag <option> para que serve o atributo "selected"?

Serve para pré-selecionar uma opção em um elemento <select>.

g) Nas tags de formulário para que serve o atributo "required"?

Serve para indicar que o preenchimento do campo associado é obrigatório para que o formulário possa ser enviado.

h) Para que serve a tag <textarea>?

Para definir um campo de entrada multilinha (uma área de texto).

i) Na tag <form> para que serve o atributo "action"?

É usado para especificar onde os dados do formulário serão enviados quando o usuário submeter o formulário.

j) Na tag <form> para que serve o atributo "method"?

Especifica o método de envio dos dados do formulário para o servidor quando o formulário é submetido. Os dois métodos mais comuns são GET e POST.

k) Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

Deve ser definido como “post”. O método “post” é considerado o mais seguro, pois não coloca as informações na URL.

l) Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?

Deve ser definido como “post” e o atributo “enctype” como “multipart/form-data".

m) O que as tags <fieldset> e <legend> produzem?

São usadas em conjunto para criar grupos de controles de formulário e fornecer uma legenda ou título descritivo para esse grupo.
